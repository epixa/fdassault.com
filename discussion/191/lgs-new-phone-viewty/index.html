  • fr4gged November 2007 Permalink
    GSM uses the SIM card techology to store subscriber information, therefore it is possible to unlock the phone to use a SIM card from another provider.

    CDMA(Code division multiple access) however, is made so that all the information is stored in the phone itself, instead of a SIM card, there is a circuit built into the phone. These phones are much more difficult to unlock, and in some cases impossible.

    90% of cellphone manufacturers always release a GSM model of a phone first, and generally if a CDMA version is what the manufacturer releases, then there isn't going to be a GSM model.
